# Shakers Semana 2 lp lotions

Este projeto faz parte do Desafio da Semana 2, com foco na criação de uma Landing Page no ambiente Shopify utilizando Liquid, Templates, Sections e Schema, seguindo boas práticas de versionamento com Git.

O objetivo do desafio não é o design, mas sim demonstrar entendimento do fluxo de desenvolvimento no Shopify, organização de código e entrega revisável pelo time.

---

## O que foi implementado

Foi criada uma Landing Page exclusiva para uma nova coleção de loções de verão, com foco em fragrâncias vibrantes e relaxantes.

A landing page conta com:

- Um produto principal em destaque (selecionável pelo Admin)
- Uma coleção de loções listada abaixo (selecionável pelo Admin)
- Destaque visual para o produto de maior valor, simulando um produto premium

Todo o conteúdo pode ser configurado pelo Editor de Temas, sem necessidade de alterar código.
